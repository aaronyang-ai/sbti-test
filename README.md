# SBTI 人格测试 · AI 重制版

全网爆火的趣味人格测试，致敬版。灵感来自 B 站 @蛆肉儿串儿 的 SBTI 测试。

## 项目介绍

SBTI（Super Brain Type Indicator）是一个纯前端的趣味人格测试网页。用户完成 31 道题目后，系统在浏览器本地计算 15 个维度的得分，匹配 27 种人格中最接近的一个，展示结果页（含雷达图 + 人格描述 + 分享卡片）。

## 技术栈

- **框架**: Next.js 14 App Router
- **语言**: TypeScript（严格模式）
- **样式**: Tailwind CSS
- **图表**: Recharts
- **图片导出**: html-to-image
- **部署**: Cloudflare Pages（静态导出）

## 数据定制

所有测试内容都存储在 `data/` 目录下，修改 JSON 文件即可定制测试内容，无需改动代码：

### 文件说明

- `data/dimensions.json` - 15 个维度的定义（代号、中文名、所属模型）
- `data/questions.json` - 31 道题目及分值映射
- `data/personalities.json` - 27 种人格的定义（代号、标签、描述、模板向量、主题色、MBTI 对照）

### 维度分值说明

每道题目的每个选项包含 `scores` 字段，记录该选项对各维度的影响分值：

- `+2` / `+1`: 正向影响，倾向高（H）
- `-1` / `-2`: 负向影响，倾向低（L）
- `0` 或不记录: 无影响

最终维度映射规则：

- 原始分 `< -1` → L（低）
- `-1 ≤ 原始分 ≤ 1` → M（中）
- 原始分 `> 1` → H（高）

## 页面结构

| 路由 | 说明 |
|-----|------|
| `/` | 首页：大标题 + 开始测试按钮 |
| `/test` | 答题页：一次显示一道题，支持上一题/下一题 |
| `/result` | 结果页：人格代号 + 雷达图 + 描述 + 分享卡片 |
| `/types` | 人格图鉴：27 种人格列表，无需答题也能浏览 |

## 匹配算法

1. 汇总 31 道题在 15 个维度的原始分
2. 原始分映射到 L/M/H 三档
3. 用户的 15 维向量与 25 个标准人格模板做曼哈顿距离比对
4. 取相似度最高的人格作为结果
5. 特殊规则：
   - 最高匹配度 < 60% → 兜底为 HHHH（傻乐者）
   - 最后一道饮酒题选择特定选项 → 触发 DRUN-K（酒鬼）隐藏结局
